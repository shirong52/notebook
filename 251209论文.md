<h1 id="《piercingeye-dual-space-video-violence-detection-with-hyperbolic-vision-language-guidance》深度解读">《PiercingEye: Dual-Space Video Violence Detection with Hyperbolic Vision-Language Guidance》深度解读</h1>
<h2 id="研究目标与问题意义">1. 研究目标与问题意义</h2>
<h3 id="研究目标">研究目标</h3>
<ul>
<li><strong>解决视觉模糊事件识别难题</strong>：针对冰球比赛中“正常碰撞”与“暴力冲突”等视觉相似但语义不同的事件，提出双空间学习框架提升判别能力。</li>
<li><strong>双曲空间优化层次建模</strong>：利用双曲空间（Hyperbolic Space）的层次特性，结合欧几里得空间（Euclidean Space）的视觉特征优势，构建互补的特征表示。</li>
<li><strong>LLM增强模糊事件监督</strong>：通过Vision-Language Model（VLM）和Large Language Model（LLM）生成模糊事件文本，补充监督信号，缓解数据稀缺问题。</li>
</ul>
<h3 id="问题意义">问题意义</h3>
<ul>
<li><strong>学术价值</strong>：首次将双曲空间与LLM结合应用于视频暴力检测，填补了“双曲几何→多模态对齐”的研究空白。</li>
<li><strong>产业影响</strong>：提升公共安全监控、自动驾驶视觉定位的鲁棒性，降低误报率；边缘设备部署时，双空间模型的轻量级设计（仅0.91MB额外参数）可实现实时处理。</li>
</ul>

<h1 id="《learning-hierarchical-hyperbolic-embeddings-for-compositional-zero-shot-learning》深度解读">《Learning hierarchical hyperbolic embeddings for compositional zero-shot learning》深度解读</h1>
<h2 id="研究目标与问题意义-1">1. 研究目标与问题意义</h2>
<h3 id="研究目标-1">研究目标</h3>
<ul>
<li><strong>层次结构建模</strong>：利用双曲空间的层次特性，建模组合零样本学习（CZSL）中的语义层次（如苹果⊂水果）和概念层次（如切片苹果⊂苹果）。</li>
<li><strong>解决层次坍塌</strong>：设计分类学蕴含损失（LTE）和判别对齐损失（LDA），避免双曲空间中的层次坍塌和细粒度区分不足问题。</li>
<li><strong>性能验证</strong>：在MIT-States、UT-Zappos、CGQA三个数据集上验证方法的有效性，超越现有CZSL方法。</li>
</ul>
<h3 id="实际问题">实际问题</h3>
<p>当前CZSL方法存在两大痛点：</p>
<ol>
<li><strong>欧几里得空间局限</strong>：多项式体积增长无法匹配层次结构的指数增长，导致层次坍塌（如父节点与子节点距离过近）。</li>
<li><strong>细粒度区分不足</strong>：传统对比损失无法有效区分语义相似的组合概念（如切片苹果与切片橙子）。</li>
</ol>
<h3 id="产业意义">产业意义</h3>
<ul>
<li><strong>降低标注成本</strong>：CZSL无需标注所有组合概念，可应用于电商推荐（如“红色连衣裙”→“蓝色连衣裙”）、医疗图像分析（如“肺癌早期”→“肺癌晚期”）等领域。</li>
<li><strong>提升泛化能力</strong>：双曲空间的层次建模增强模型对未见过组合概念的识别能力，适用于数据稀缺场景。</li>
</ul>

<h2 id="论文不足与存疑之处-1">5. 论文不足与存疑之处</h2>
<h3 id="不足">不足</h3>
<ol>
<li><strong>依赖预定义层次</strong>：当前方法需要人工或LLM生成层次结构，无法处理动态变化的层次。</li>
<li><strong>计算复杂度</strong>：双曲空间的指数映射和距离计算比欧几里得空间更耗时。</li>
<li><strong>数据集局限性</strong>：在CGQA数据集上的AUC仅16.0%，性能提升空间较大。</li>
</ol>
