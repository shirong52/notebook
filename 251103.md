<h3 id="、不同的子序列">1、不同的子序列</h3>
<ul>
<li>dp[i][j]：以i - 1的s中有j - 1为结尾的t的个数</li>
<li>if(s[i - 1] == t[j - 1]): dp[i][j] = dp[i - 1][j - 1] + dp[i - 1][j]（考虑使用和不使用两种情况）else dp[i][j] = dp[i - 1][j]（把s这个元素删除掉）</li>
<li>dp[i][0] = 1（s中有一种为空字符串t的方式）,dp[0][j] = 0（空字符串s有0种为t的方式），dp[0][0] = 1（上面两种情况的交集）</li>
<li>从左到右从上到下</li>
</ul>
<h3 id="、两个字符串的删除操作">2、两个字符串的删除操作</h3>
<ul>
<li>dp[i][j]：以i - 1为结尾的word1、以j - 1为结尾的word2相同的最小操作次数</li>
<li>if(word1[i - 1] == word2[j - 1]) dp[i][j] = dp[i - 1][j - 1];else dp[i][j] = min(dp[i - 1][j] + 1, dp[i][j - 1] + 1, dp[i - 1][j - 1] + 2)</li>
<li>dp[i][0] = i， dp[0][j] = j（word1为空，word2要删除j个元素和其相同）</li>
<li>从左到右从上到下</li>
</ul>
<h3 id="、编辑距离">3、编辑距离</h3>
<ul>
<li>dp[i][j]：以i - 1为结尾的word1、以j - 1为结尾的word2相同的最小操作次数</li>
<li>if(word1[i - 1] == word2[j - 1]) dp[i][j] = dp[i - 1][j - 1]（不需要操作）;else dp[i][j] = min(dp[i - 1][j] + 1, dp[i][j - 1] + 1, dp[i - 1][j - 1] + 1)（前一个相同的情况下这一个不同需要进行替换）</li>
<li>dp[i][0] = i, dp[0][j] = j（要将其操作为空串）</li>
<li>从左到右从上到下</li>
</ul>
